Version 2.0.0 — 23 Aug, 2024

Folio v2.0 introduces a complete overhaul with a fresh design system and is built on the latest Bootstrap v5. This version is entirely reimagined and developed from the ground up. Please note that migrating from v1.x to v2.x is not supported.
