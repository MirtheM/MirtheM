Font icons

Retina-ready font icons with a modern touch that will beautifully complement your project.

We have included Bootstrap Icons and Iconstica in the Folio theme. Below are the instructions on how to use them effectively.

Bootstrap Icons:

Bootstrap Icons are also included in the Folio theme. Similar to Font Awesome, Bootstrap Icons can be used by adding specific class names.

Add icons to your HTML

Use the <i> tag with the appropriate class names to add icons:

	
	<i class="bi bi-alarm"></i>

A complete list of Bootstrap icons along with the class names can be found here

Iconstica Icons:

In Folio, we've incorporated free Iconstica icons. These icons offer a wide variety of high-quality options for different purposes. Check out the free icons here.
